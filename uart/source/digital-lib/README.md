# SoCET Digital Library
Parameterizable, re-usable implementations of common RTL submodules.

Consult the `doc/` folder for specific information about each module.

## FuseSoC Setup
This library is intended to be used with the [FuseSoC](https://fusesoc.net/) build system. Each submodule
has its own `.core` file to use with FuseSoC. To make use of this library, you can either clone it or add it as
a git submodule to your project, or use `fusesoc library add` to add this library to your project.
